# Contribuindo

Obrigado por contribuir com a ZapForge API.

## Ambiente

```bash
cp .env.example .env
npm install
npm run dev
```

## Padrões

- TypeScript strict.
- Rotas em `src/routes`.
- Regras de negócio em `src/core`.
- Engines em `src/adapters`.
- Não coloque segredos em commits.
- Não adicione recursos voltados a spam ou abuso.

## Pull request ideal

1. Descreva o problema.
2. Explique a solução.
3. Mostre como testar.
4. Atualize a documentação se a API mudar.
